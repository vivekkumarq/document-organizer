# Smart Digital Document Organizer

A full-stack application to securely manage personal documents.

## Tech Stack

Backend
- Spring Boot
- PostgreSQL
- JWT Authentication
- BCrypt security

Frontend
- React (Vite)
- Axios

## Features

- User registration & login
- Secure document upload
- Document listing
- File download
- JWT secured APIs

## Architecture

React → Spring Boot → PostgreSQL + File Storage

## Run Backend

mvn spring-boot:run

## Run Frontend

npm install
npm run dev
